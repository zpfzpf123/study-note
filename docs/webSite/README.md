# Vue相关
## Vue
Vue 是一款流行的 JavaScript 前端框架，用于构建用户界面。它采用组件化的开发模式，易于上手且灵活性高，适用于各种规模的项目。
### Vue2
[Vue2 官方指南](https://v2.cn.vuejs.org/v2/guide/)：Vue2 的官方文档，详细介绍了 Vue2 的基础语法、组件系统、指令等核心功能，是学习和使用 Vue2 的重要参考资料。
### Router(Vue2)
[Vue Router 3.x 安装指南（Vue2 适用）](https://v3.router.vuejs.org/zh/installation.html)：Vue Router 是 Vue.js 官方的路由管理器，此链接为适用于 Vue2 的 Vue Router 3.x 版本安装及使用指南，帮助开发者实现单页面应用（SPA）的路由功能。
### Vuex
[Vuex 3.x 官方文档（Vue2 适用）](https://v3.vuex.vuejs.org/zh/)：Vuex 是专为 Vue.js 应用程序开发的状态管理模式，该文档针对 Vue2 讲解了 Vuex 的使用，通过集中式存储管理应用的所有组件的状态，使状态变化可预测。
### Vue3
[Vue3 官方指南](https://cn.vuejs.org/guide/introduction.html)：Vue3 在性能、功能等方面有诸多提升，此文档介绍了 Vue3 的新特性、使用方法等内容，帮助开发者快速上手 Vue3 开发。
### Router(Vue3)
[Vue Router 4.x 官方文档（Vue3 适用）](https://router.vuejs.org/zh/introduction.html)：适配 Vue3 的 Vue Router 4.x 版本官方文档，提供了在 Vue3 项目中使用路由功能的详细指导。
### Pinia
[Pinia 官方文档](https://pinia.vuejs.org/zh/introduction.html)：新一代的 Vue 状态管理库，相较于 Vuex 更加简洁易用，此文档介绍了 Pinia 的使用方法和特性。

# UI框架相关
## UI
UI 框架为开发者提供了一系列预定义的 UI 组件，可加快项目开发速度，提升用户界面的一致性和美观度。
### ElementUI(Vue2)
[ElementUI 2.x 安装指南（Vue2 适用）](https://element.eleme.cn/#/zh-CN/component/installation)：一套为开发者、设计师和产品经理准备的基于 Vue2 的桌面端组件库，提供丰富的组件，如按钮、表单、表格等，助力快速搭建美观且功能丰富的界面。
### ElementUI(Vue3)
[Element Plus 官方文档（Vue3 适用）](https://element-plus.org/zh-CN/component/overview.html)：基于 Vue3 开发的桌面端组件库，是 ElementUI 的升级版本，继承了 ElementUI 的设计风格和使用习惯，并针对 Vue3 的特性进行了优化。
### iView
[iView 4.x 官方文档](http://v4.iviewui.com/docs/introduce)：一套基于 Vue.js 的高质量 UI 组件库，提供了大量美观实用的组件，适用于开发中后台产品界面，注重用户体验和开发效率。
### Ant Design Vue
[Ant Design Vue 官方文档](https://www.antdv.com/components/overview-cn)：基于 Ant Design 设计体系的 Vue 实现，拥有丰富的组件和良好的设计规范，适用于企业级中后台应用的界面开发。
### DataV
[DataV 官方指南](http://datav.jiaminghi.com/guide/)：专注于数据可视化的工具，提供了丰富的可视化组件和模板，方便开发者快速搭建数据可视化大屏等应用，适用于展示各类业务数据。
### fast - dataview - ui(Vue3)
[fast-dataview-ui官方文档（Vue3 适用](https://aafx11.github.io/fast-dataview-ui/components/scrollTable.html)：一个适用于 Vue3 的数据展示 UI 库，提供了如滚动表格等特色组件，帮助开发者快速实现数据展示功能。

# 时间处理相关
## 时间
在开发中，经常需要对时间进行处理，以下是两个常用的时间处理库。
### moment
[moment.js 中文文档](http://momentjs.cn/docs/)：一个功能强大的 JavaScript 时间库，提供了各种时间格式化、解析、操作等功能，能轻松处理不同格式的时间数据，在前端和后端 JavaScript 开发中广泛应用。
### dayjs
[dayjs 安装指南](https://dayjs.gitee.io/docs/zh-CN/installation/installation)：轻量级的 JavaScript 时间库，与 moment.js 语法类似，但体积更小，专注于提供简洁的时间处理功能，适用于对性能和体积要求较高的项目。

# HTTP 相关
## HTTP
在前端开发中，与服务器进行数据交互是常见需求，axios 是一个常用的 HTTP 库。
### axios
[axios 中文文档](https://www.axios-http.cn/docs/intro)：一个基于 Promise 的 HTTP 库，可用于浏览器和 Node.js，能轻松实现 HTTP 请求和响应处理，支持拦截请求和响应、取消请求等功能，是前端开发中进行数据交互的重要工具。

# 可视化组件库相关
## Echarts
可视化组件库用于将数据以图表等可视化形式展示，方便用户理解和分析数据。
### Echart官网
[Echarts 官方网站](https://echarts.apache.org/zh/index.html)：Echarts 是一个由百度开源的数据可视化库，提供了丰富的图表类型和交互功能，适用于各类数据可视化场景，从简单的统计图表到复杂的地理信息图等都能轻松实现。
### MCChart
[MCChart 网站](http://echarts.zhangmuchen.top/#/index)：基于 Echarts 二次开发的图表库，在 Echarts 的基础上进行了封装和扩展，提供了一些更便捷的使用方式和个性化的图表样式。
### chartLib
[chartLib 网站](http://chartlib.datains.cn)：提供多种数据可视化组件和工具，帮助开发者快速实现不同类型的数据可视化需求，支持多种数据格式和交互方式。
### ppchart
[ppchart 网站](https://ppchart.com/#/)：一款简单易用的数据可视化工具，专注于提供简洁直观的图表生成功能，适合初学者快速上手制作基本的数据图表。
### v - chart2
[v - charts2 官方文档](https://denaro-org.github.io/v-charts2/charts/)：基于 Vue 和 Echarts 封装的图表组件库，以组件化的方式使用 Echarts，方便在 Vue 项目中快速集成各种图表，减少开发工作量。
